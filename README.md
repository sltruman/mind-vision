### 获取相机视频流

```bash
$ mind-vision open <产品序列号> #产品序列号:'041032420559'
'True MV-SUA630C#0 \n'	#返回值 unix-domain套接字连接地址

#套接字输入
unix-domain-socket>'next\n'
#套接字输出
'3000 2000 3 \n' 
<二进制数据流>
```

### 曝光

```bash
$ mind-vision open 产品序列号
> exposure
#返回值 曝光模式 亮度范围最小 亮度范围最大 亮度 抗频闪功能 消频闪的频率选择 模拟增益值范围最小 模拟增益值范围最大 模拟增益值 曝光时间
```


### 参数保存/读取

```bash
$ mind-vision open 产品序列号
> params-reset
'True \n'

> params-save 0  #'0|1|2|3'
'True \n' 

> params-load 0  #'0|1|2|3'
'True \n'

> params-save-to-file 'path/to/file' #文件路径
'True \n'

> params-load-from-file 'path/to/file' #文件路径
'True \n' 
```
